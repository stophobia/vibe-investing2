# AMQS-AI-Infra 매수 시그널 봇 (Signal_Bot)

AMQS-AI-Infra 전략의 **매수 시그널**을 매일 계산하고, 간단한 웹뷰 대시보드로 보여주는 봇이다.
상위 전략 엔진([`../script/strategy.py`](../script/strategy.py))을 그대로 호출하므로 백테스트와
동일한 로직(4-Factor 모멘텀 + 단기 하락 매수 + Top-N 선별 + 서브테마 캡 + 거시 레짐)으로 신호를 낸다.

![매수 시그널 대시보드](assets/dashboard_top.png)

## 무엇을 보여주나

- **거시 레짐 배너** — Risk-On / Risk-Off / Defensive 와 투자 비중
- **매수 시그널 카드** — Top-N 에 선별된 종목의 목표 비중·점수·티어(중심/위성/전술 매수)
- **단기 하락 매수(DIP) 알림** — 추세는 살아있고 단기 조정 중인 종목(예: INTC)
- **전체 종목 점수표** — 19종 모멘텀·RSI·52주 위치·시그널
- **하단 리스크 고지** — 고변동·고베타·인샘플 특성 등 명시 (웹사이트 하단 필수 고지)

## 구성

| 파일 | 역할 |
|---|---|
| `signal_bot.py` | 전략 엔진 호출 → `signals.json` / `signals.js` 출력 |
| `index.html` | 정적 웹뷰 대시보드 (매수 시그널 강조 + 하단 리스크 고지) |
| `signals.json` | 봇이 생성한 최신 시그널 |
| `signals.js` | `file://` 직접 열기용 JS 래퍼 (브라우저 로컬 fetch 차단 회피) |

## 실행

```bash
# 상위 전략의 의존성 설치
pip install -r ../requirements.txt

# 실데이터로 매수 시그널 생성
python signal_bot.py --period 2y --outdir .

# 네트워크 없이 로직만 검증 (합성 데이터)
python signal_bot.py --mock --outdir .
```

## 웹사이트로 보기

- **간단히**: `index.html` 을 브라우저로 직접 열기 (`signals.js` 로 데이터 주입)
- **로컬 서버**: `python -m http.server 8011` 후 `http://localhost:8011/`
- **GitHub Pages**: 이 폴더를 Pages 로 게시하면 그대로 대시보드

## 매수 시그널 정의

`CENTER`(≥80) · `SATELLITE`(65~80) · `TACTICAL`(50~65) · `DIP_BUY`(단기 하락 매수) 중
Top-N(기본 10, 서브테마당 최대 4)에 선별되어 목표 비중 > 0 인 종목을 **매수 후보**로 표시한다.
거시 레짐이 Risk-Off 면 50%만, Defensive 면 방어 바스켓으로 전환된다.

## 리스크 고지

본 봇과 결과는 통계·기술적 모멘텀 신호에 기반한 **연구·교육용 보조 지표이며, 특정 종목의 매매를
권유하지 않는다.** AI 인프라는 고변동·고베타 섹터로 종목 간 상관이 높아 분산 효과가 제한적이다.
모든 수치는 인샘플 강세장 특성과 레짐 의존성을 품고 있어 그대로 미래에 작동한다는 보장이 없으며,
실제 투자에는 **원금 손실을 포함한 상당한 리스크**가 따른다. 거래비용·세금·환전·슬리피지는
별도이며, 모든 투자 판단과 그 결과의 책임은 사용자 본인에게 있다.
