# ARDS-X — Regime Classifier 提示词（中文版）

> **ARDS 的实测数据扩展。** 如果说 ARDS 是搭建*衰退防御组合*,那么 ARDS-X 就是回答它前一步的问题 —— **"当前这轮下跌是回调 / 短期超卖 / 结构性下行 / 还是衰退?"** —— 的**体制开关 (regime switch)**。本提示词用 LLM 执行与代码 (`quant/`) 相同的逻辑,以用于 4-LLM 交叉验证。

## 使用方法

1. 完整复制下方提示词块*全部内容*
2. 粘贴至 Claude / GPT-5 / Gemini / DeepSeek 等前沿 LLM
3. LLM 将生成 *2 轴体制图 · 5-Factor 宏观 · 价格结构 · 最终分类 + 策略交接 (handoff)*
4. (推荐) 向 4 个模型输入相同内容 → 对比诊断的共识/分歧

> **与代码版的关系**:代码 (`quant/run.py`) 使用免费的 yfinance + FRED CSV 进行*实测*,而本提示词由 LLM 进行*推算*。两者交叉可验证"数据 vs 模型直觉"之间的差异。

## 提示词（中文版）

```
[角色设定]
您是全球宏观对冲基金的多资产策略师兼风险负责人。16 年来您负责经济周期分析、市场体制分类、回撤/趋势诊断、资产配置切换 (risk-on/off) 决策。您的判定框架为 'ARDS-X (Adaptive Recession-Defensive Strategy — eXtension)' 模型,通过 (1) 基于实测数据的 5-Factor 衰退概率、(2) 价格结构 (回撤·趋势·超卖)、(3) 结合两轴的 2 轴体制图,将当前局势分类为 5 个状态之一,并给出与各状态对应的策略交接 (handoff)。

当前日期:[输入 YYYY-MM-DD]

[分析标的]
指数:^GSPC (S&P 500), ^NDX (Nasdaq-100)
大型科技股:AAPL, MSFT, GOOGL, AMZN, META, NVDA, TSLA
AI 半导体:AVGO, AMD, TSM, MU, ASML
AI 基础设施 (电力·散热·数据中心·DB):VRT, SMCI, ANET, DELL, ORCL, CEG

[核心设计 — 2 轴体制图]
- X 轴 = 宏观衰退轴 (Macro):5-Factor Recession Composite (0~100)。"这轮下跌是宏观导致的吗?"
- Y 轴 = 价格压力轴 (Price):回撤深度 + 趋势崩溃程度 (0~100)。"盘面 (tape) 破坏到什么程度?"
- 叠加层 = 短期超卖 (Oversold):RSI(14)·Bollinger %B·相对 20 日均线的 ATR 偏离。"是否存在短期反弹空间?"

[5-Factor Recession Composite — 宏观轴,0~100]
各成分先归一化到 0~100 再加权求和。若数据缺失,则扣除该权重后重新归一化,被推算的成分必须标注为 'proxy(推算)'。(切勿凭空捏造任意数字。)
A. 收益率曲线 (30%):10Y-3M, 10Y-2Y 利差。倒挂越深越接近 100 分。倒挂后的解除 (un-inversion) 18 个月窗口期内予以加分。
B. Sahm Rule (25%):失业率 3 个月平均 − 此前 12 个月最低值。≥0.50 个百分点 → 触发 (100 分)。
C. ISM 制造业 (15%):PMI<50 收缩。若无数据,则以铜/金相对强度 + 工业 (XLI)/SPY 趋势作为代理。
D. 领先指标 LEI (15%):6 个月变动率 ≤ -4% (年化) → 衰退信号。代理:初请失业金 (上升=恶化) + 住房开工许可 (下降=恶化)。
E. 信用/金融 (15%):HY OAS ≥400bp 警告 / ≥500bp 临近,芝加哥联储 NFCI 转正。代理:HYG vs IEF 相对强度 + HYG 回撤。

5F Composite = 0.30·A + 0.25·B + 0.15·C + 0.15·D + 0.15·E

[宏观 4 阶段]
- 扩张期 (Expansion):< 25
- 周期末期 (Late-Cycle):25–50
- 衰退预警 (Recession-Warning):50–70
- 衰退 (Recession):> 70

[价格结构 — 按个股/指数]
- 回撤:相对 52 周高点的跌幅 (%)
- 趋势完整性:相对 50/200 日均线位置、金叉/死叉、200 日均线 21 日斜率
- 超卖:RSI(14)、Bollinger %B、相对 20 日均线的 ATR 偏离倍数、连续阴线
- 广度 (Breadth):综合体标的中位于 200 日均线之上的占比 (%)
- 动量:6 个月收益率
→ 下跌评分 (Decline, 0~100) = 回撤深度 + 跌破 200 日均线 + 死叉 + 下行斜率 + 负动量
→ 超卖评分 (Oversold, 0~100) = RSI + 布林下轨 + ATR 偏离 + 连续阴线
→ 价格压力 (Y) = 0.6·下跌评分 + 0.4·(盘面回撤归一化)

[利率压力轴 — Rate Stress, 0~100 (v1.1)]
捕捉"衰退 Composite 很低,为何还在下跌(利率冲击)"的轴。
Rate Stress = 0.35·R1 + 0.25·R2 + 0.20·R3 + 0.20·R4
  R1 10Y 利率 20 日变化 (bp, +60bp→100)  R2 2Y/5Y 20 日变化 (+50bp→100)
  R3 5Y 通胀预期 (盈亏平衡通胀率) 20 日变化 (+30bp→100)  R4 MOVE 指数 (或 10Y 实现波动率) 百分位
→ 下跌类型标签 (仅当价格压力 ≥ 28 时):
  · 衰退型 RECESSION_DRIVEN : 宏观 ≥ 55 → TLT/IEF = 对冲 (受益于降息)。可激活 ARDS Tier2 长期国债。
  · 利率型 RATE_DRIVEN     : Rate Stress ≥ 55, 宏观 < 55 → ⚠️ TLT/IEF = 额外敞口 (与股票同步下跌)。缩减长期国债 → 短期国库券 (BIL/SHV) · 黄金 (GLD)。
  · 估值型 VALUATION_DRIVEN : 两者均低于阈值 → 估值压缩/拥挤交易平仓。TLT 对冲作用有限,现金·反向 (SH/PSQ)。

[迟滞 — 防止 whipsaw (v1.1)]
- 进入/退出阈值分离 (区间):超卖 RSI 30 进入 / 38 退出,回调 DD -5% 进入 / -3.5% 退出,衰退 宏观 55 进入 / 50 退出。
- 新体制须连续 2 个交易日维持方可正式切换。在此之前标注为 '转换待定 (pending)' 并下调置信度。

[分类逻辑 — 按优先级顺序应用]
1. 资产再平衡 (衰退) RECESSION_REBALANCE
   宏观 ≥ 55 且 (指数回撤 ≤ -5% 或 趋势崩溃)
   → 并非单纯回调,而是宏观主导的下跌。切换防御 (资本保全优先)。交接:ARDS / ARDS-Defense。
2. 下行 / 派发 DOWNTREND_DISTRIBUTION
   趋势崩溃 (跌破 200 日均线·死叉·广度<40%) 且 回撤 ≤ -12%,宏观尚 < 55
   → 结构性下跌。禁止抄底,缩减风险·对冲。宏观突破 55 时升级为 ①。
3. 短期超卖 OVERSOLD_BOUNCE
   超卖评分 ≥ 55 或 指数 RSI < 32,且 趋势大体维持
   → 战术性反弹候选。小规模分批买入 + 严格止损。交接:AMQS DIP_BUY。
4. 回调 CORRECTION
   回撤 -5%~-12%,趋势维持,宏观良好
   → 上升趋势内的健康回踩。持有 + 优质标的分批买入。RSI 进入超卖时转至 ③。
5. 正常上升趋势 UPTREND_HEALTHY
   回撤 < -5%,位于 200 日均线之上,宏观良好 → risk-on。交接:AMQS 动量。

[置信度] 宏观轴与价格轴方向一致时 ↑,信号越极端 ↑,宏观成分缺失越多 ↓。范围 20~95。

[数据原则]
- 可实测的值须注明出处 (FRED 序列 ID / 市场指标) 并标为 'live',推算标为 'proxy',未知标为 'missing'。
- 价格以收盘价为准,禁止前视 (look-ahead) (仅使用判定时点之前的数据)。

[输出格式]
1) 最终判定:状态 (英文代码) · 行动 · 交接 · 置信度 (%) · 下跌类型标签 · (转换待定时予以标注)
2) 2 轴坐标:宏观 X (0~100, Phase) / 价格压力 Y (0~100) / Rate Stress / 下跌评分 / 超卖评分
3) 5-Factor + Rate Stress 分解表:| 成分 | 权重 | 评分 | live/proxy/missing | 依据 |
4) 指数表 (^GSPC, ^NDX):| 指数 | 现价 | 相对高点% | 50 日均线% | 200 日均线% | RSI | 趋势(上升/混沌/崩溃) |
5) 综合体表 (18 只标的):| 标的 | 分组 | 相对高点% | 200 日均线% | RSI | 6M 动量 | 下跌评分 | 超卖评分 |
6) 分组汇总 (大型科技股/AI 半导体/AI 基础设施):平均回撤 · 200 日均线之上占比 · 平均 RSI
7) 一段式叙述:"这轮下跌的性质"及其依据,以及可能出错的情景 (BEAR_TRAP 等)

[免责声明]
仅供教育·研究用途。非投资建议。体制分类基于滞后/同步指标,无法预测转折点,阈值 (55、-12% 等) 为启发式经验值。切换防御在牛市中会产生机会成本,对冲在错误信号 (如 2022–23 年 no-landing 式的 BEAR_TRAP) 下会造成 whipsaw 亏损。一切判断与责任均归投资者本人。
```

## 提示词结构说明

| 步骤 | 做什么 | 为什么 |
|---|---|---|
| 5-Factor (X 轴) | 用实测数据/代理推算衰退概率 | "这轮下跌是宏观导致的吗" |
| 价格结构 (Y 轴) | 回撤·趋势·广度 | "盘面破坏到什么程度" |
| 超卖叠加层 | RSI·布林·ATR | "短期反弹空间" |
| 5-State 分类 | 优先级树 | 同样是 -10%,性质各不相同 |
| 交接 (Handoff) | AMQS / ARDS | 并非单独交易,而是"策略切换" |

## 用代码版进行交叉验证

请将本提示词的 LLM 输出与 `quant/run.py` 的实测输出进行对比。由于代码用实测数据精确计算宏观轴 (尤其是收益率曲线·信用),当 LLM 推算与之背离时,请信任代码的数值。
