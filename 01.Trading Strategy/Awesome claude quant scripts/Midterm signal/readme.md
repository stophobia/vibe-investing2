# 미국 중간선거 해 조정장 매수 시그널 점검 프롬프트

**Midterm Election Dip-Buying Signal Checker**
> 작성 2026-05-24 · 투자 권유가 아니며 교육·참고 목적입니다.
> 동봉된 `README.md`(전략 배경·철학·백테스트)와 `midterm_backtest.py`(검증 코드)와 함께 사용하세요.

이 투자 전략은 미국 중간 선거 해에 조정장이 오면 매수를 하라는 오랜 투자 격언에 따른 LLM 프롬프트입니다.
결론적으로 이 전략은 집권당이 선거에 패배하게 되면 과감한 경제 정책을 통해 차기 정권을 창출하기 위한 재정정책이 있는 점을 포인트로 삼고, 앙드레 코스톨라니의 페타 꼼플리(Fait Accompli), 선거 패배 후 불확실 개선이 그 심리적인 포인트입니다.
다만, 지금 인공지능 혁명으로 인해 인공지능과 M7의 차별화된 시장이 있어 2026년 올해 조정장이 올지는 모르겠습니다.

## 0. 검증 요약 (백테스트 기반 팩트체크)

`midterm_backtest.py`로 1938~2022년 22번을 직접 검증한 결과:

| 항목             | 원본 이미지 주장 | 직접 백테스트 / 기관 검증               | 판정       |
| -------------- | --------- | ----------------------------- | -------- |
| 1년 뒤 평균 수익률    | +36.4%    | **+19.11%** (연말 매수, 실행 가능 기준) | 과장     |
| 승률(플러스)        | 사실상 전부 +  | **95.5%** (21승 1패)            | 대체로 사실  |
| 초과수익 (vs 다른 해) | (언급 없음)   | **+9.01%p** (다른 해 +10.1% 대비)  | 유의미     |
| 유일한 실패         | (언급 없음)   | **1939년 (대공황+2차대전)**          | 함정     |
| 통계적 유의성        | (언급 없음)   | T-검정 결과 **유의하지 않음**           | 약점     |

**핵심:** "1년 뒤 반등"은 진짜지만 +36%는 "저점 매수"를 가정한 상한선이다.
현실적 기대치는 +19% 수준. 그리고 95.5% 승률은 "12개월 측정 구간"의 함정을 품고 있어
구조적 위기(1939)나 secular 약세장(1966·1970·1974)에서는 깨진다.

## 1. 복사해서 쓰는 프롬프트 (Copy & Paste)

```
# 미국 중간선거 해 조정장 매수 시그널 점검 프롬프트

당신은 매크로/퀀트 관점의 투자 분석가다. 아래 절차에 따라
"중간선거 해 하락 시 매수" 전략의 현재 적용 가능성을 점검하라.
감정적 낙관이나 과거 데이터 맹신을 배제하고, 반증 가능한 근거만 사용하라.
필요한 모든 수치는 웹 검색으로 최신값을 확보하라.

## STEP 0 — 연도 판별
- 올해가 미국 중간선거 해(짝수 연도 중 대선이 없는 해)인지 확인.
- 아니라면: "올해는 중간선거 해가 아님. 이 전략 비적용" 답하고 종료.

## STEP 1 — 전제 검증 (가장 중요, 항상 답변 맨 앞 1~2줄)
- "1년 뒤 +36%"는 연중 최저점 매수를 가정한 사후적·이론적 수치.
  실제 실행 가능(연말 매수) 기준 백테스트 평균은 약 +19%.
- 승률은 높으나(약 95%) 이는 12개월 측정 구간의 함정을 포함.
  1939년(공황·전쟁)에는 실패했고, 1966/70/74는 단기 반등 후 약세장 지속.
- 중간선거 효과는 표본이 작아(약 22~31회) 통계적으로 유의하지 않다.

## STEP 2 — 시장 상태 진단 (검색으로 실데이터 확보, 표로 정리)
1. S&P 500 / 나스닥100: 연초 대비 및 최고점 대비 낙폭(%)
2. 밸류에이션: S&P 500 선행 PER (장기 평균 18.8배 대비)
3. 금리/유동성: 연준 정책금리, 10년물 금리, 인하/인상 기대
4. 기업 실적: 직전 분기 EPS 증가율, 어닝 서프라이즈, 가이던스
5. 심리/변동성: VIX 수준, Fear & Greed
6. 폭(Breadth): 광범위한 하락인가, 소수 종목 문제인가
7. (보너스) 주식 위험 프리미엄 — 0에 가까우면 안전마진 없음

## STEP 3 — 시그널 판정 (각 충족 시 +1)
[ ] 최고점 대비 -10% 이상 조정 발생
[ ] 밸류에이션이 장기 평균 이하로 정상화
[ ] 금리 인하 사이클 진입 또는 동결
[ ] 기업 EPS가 여전히 증가 (이익 침체 아님)
[ ] 하락 원인이 거시 공포이지 구조적 실적 훼손이 아님
[ ] VIX 급등 등 단기 패닉 신호
판정: 5~6점=매수 우호(분할) / 3~4점=중립 / 0~2점=매수 금지

## STEP 4 — 실행 가이드
- 일시 매수 금지. 분할매수(예: 4~6주 분할) 전제로만 제안.
- "왜 하락하는가" 한 문장 결론. (정치 노이즈=기회 / 구조적=회피)
- 반대 시나리오(이번엔 다를 수 있는 이유) 최소 1개 명시.

## 출력 형식
1. 한계 경고 (1~2줄)
2. 시장 진단 표
3. 시그널 점수 / 판정
4. 실행 코멘트 + 반대 시나리오

```

## 2. 2026년 실행 결과 (2026-05-24 기준)

> ⚠️ "+36%"는 저점 매수 전제의 이론치(현실 기대치 약 +19%). 중간선거 효과는 통계적으로 유의하지 않음. 투자 권유 아님.

| 지표         | 현재 상태                                     | 시그널    |
| ---------- | ----------------------------------------- | ------ |
| S&P 500 위치 | 약 7,473 (5/22), 사상 최고가 0.5%p 이내, 연초 +1.5% | 조정 미발생 |
| 밸류에이션      | 선행 PER 22.1배 (10년 평균 18.8배 대비 고평가)        | 비쌈     |
| 금리         | 정책금리 3.50~3.75%, 상반기 추가 1회 인하 전망          | 우호     |
| 실적         | 약 78% 컨센서스 상회 (10년 평균 74%)                | 견조     |
| 변동성        | VIX 대체로 14~22 안정 범위                       | 패닉 없음  |
| 위험 프리미엄    | ERP 약 0.02%, 역대 최저 (안전마진 없음)              | 경고     |

**시그널 점수: 2 / 6 → "매수 금지" 구간**

2026년은 중간선거 해이고 금리·실적은 우호적이나, **시장이 아직 하락하지 않았고 오히려 고평가**다.
이 전략은 "하락 시 매수"이므로 전제 조건 자체가 미충족. 1980년 이후 연중 최고점 대비 중간값
낙폭은 10.4%, 2026년 중 10%+ 하락 확률은 약 50%. **조정이 온다면** 이 점검표를 다시 돌려
점수가 5~6점으로 오르는지 확인하는 것이 본래 용도다.

**반대 시나리오:** 위험 프리미엄이 역대 최저라는 점은, 조정이 와도 단순 반등이 아니라
밸류에이션 정상화(더 깊고 긴 하락)로 이어질 수 있음을 시사한다.

## 3. 3개 LLM 교차검증 (같은 프롬프트, 같은 날)

> 같은 프롬프트를 2026-05-24에 ChatGPT · Gemini · DeepSeek 세 모델에 동시 실행한 결과.
> 멀티-LLM 교차검증(multi-LLM cross-validation)으로 단일 모델 편향을 줄이고 합의/불일치를 드러낸다.

![3개 LLM 교차검증 비교](llm_comparison.png)

### 3-1. 결과 요약

| 모델 | 점수 | 최종 판정 | 핵심 논거 |
|------|------|-----------|-----------|
| **ChatGPT** | 2 / 6 | 매수 금지 ~ 중립 하단 | AI 고밸류 + 금리 부담, 정치 이벤트보다 구조적 특수성 |
| **Gemini** | 3 / 6 | 중립 | ③ 금리 인하 사이클을 충족으로 판단 (유일하게 3점) |
| **DeepSeek** | 2 / 6 | 매수 금지 | 가장 구체적 수치, 이란發 고유가·금리 인상 우려 |

**합의:** 세 모델 모두 **"지금은 매수 시점이 아니다"** 로 수렴 (2~3점).
직접 백테스트 기반 본 README의 판정(2/6)과도 일치한다.

### 3-2. 공통 충족 / 공통 미충족 항목

| 항목 | ChatGPT | Gemini | DeepSeek | 합의 |
|------|:---:|:---:|:---:|------|
| ① 최고점 대비 -10%↑ 조정 | ✕ | ✕ | ✕ | **전원 미충족** (조정 자체가 없음) |
| ② 밸류에이션 정상화 | ✕ | ✕ | ✕ | **전원 미충족** (고PER 지속) |
| ③ 금리 인하/동결 | ✕ | ✓ | △ | **불일치** (금리 전제 차이) |
| ④ 기업 EPS 증가 | ✓ | ✓ | ✓ | **전원 충족** (실적 견조) |
| ⑤ 하락 원인 비구조적 | ✓ | ✓ | ✓ | **전원 충족** (구조적 훼손 아님) |
| ⑥ VIX 패닉 신호 | ✕ | ✕ | ✕ | **전원 미충족** (VIX 안정) |

### 3-3. 핵심 불일치 — 금리 전제 (③)

세 모델의 점수 차이는 거의 전적으로 **금리(③) 항목 하나**에서 갈렸다.

- **Gemini:** 기준금리를 4.50~4.75%·인하 사이클 진행 중으로 보고 ✓ → 3점
- **DeepSeek:** 정책금리 3.50~3.75% 동결이나 **인상 전환 우려**로 보고 △ → 2점
- **ChatGPT:** 고금리 유지·인하 기대 후퇴로 보고 ✕ → 2점

→ 같은 날 같은 프롬프트인데도 **금리 수준·방향에 대한 인식이 모델마다 달랐다.**
이는 모델별 학습 데이터 최신성과 추론 방식의 차이를 보여주는 사례이며,
**단일 LLM의 거시 수치를 그대로 신뢰하면 안 된다**는 교훈을 준다.
(실측 기준 2026년 5월 정책금리는 3.50~3.75% 수준 — DeepSeek/ChatGPT의 레벨 인식이 더 근접)

### 3-4. 합의가 주는 신뢰도

세 모델이 **독립적으로 모두 "매수 금지~중립"** 에 도달했다는 점은 의미가 크다.
서로 다른 학습 데이터·아키텍처를 가진 모델들이 같은 결론에 수렴하면,
그 판정이 특정 모델의 편향이 아닐 가능성이 높아진다.
다만 ④⑤(실적 견조·비구조적)에 모두 ✓를 준 것은,
**조정이 실제로 오기만 하면 전략 발동 조건이 빠르게 갖춰질 수 있다**는 뜻이기도 하다.
