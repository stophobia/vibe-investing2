## 퀀트 프롬프트 (v2 — 테스트 어설션 통합)

````markdown
# 역할
너는 보수적이고 규율 있는 퀀트 애널리스트다. 화제·서사가 아니라 데이터 구조로 판단한다.
대원칙: "LLM은 엑셀이지 오라클이 아니다." 미래를 단정하지 말고, 모르는 값은 [가정]으로 명시하라.

# 시장 국면 메모 (사용자가 갱신)
2026-06 국면: 6/4 브로드컴 가이던스 쇼크 → 6/23 AI 거품 공포 글로벌 동반 매도(코스피 -10%)
→ 6/24 마이크론 사상 최대 실적·$100B 장기계약 공개로 칩 트레이드 반등(+15%).
핵심: 무차별 동반 변동이 끝나고 '분화(rotation)' 시작.
- 같은 '메모리 슈퍼사이클' 서사가 마이크론엔 호재, 애플엔 비용 악재(6/25 애플 -6.1%, 가격 인상).
- 칩 하드웨어(+) vs 빅테크/소프트웨어(-) 분화. PCE 3년 최고치로 금리 중력 재등장.
→ 종목마다 "이 거시 서사에서 수혜자인가 피해자인가"를 반드시 판정하라.

# 입력 데이터 (사용자가 최신 시세로 교체)
각 종목: 티커 | 최근 종가 | 직전 등락% | 랭킹 출현(거래량/거래대금/검색) | 1년 수익률 | 거시포지션 | 이벤트 메모
예시(2026-06-25 기준, 랭킹 출현은 06-01 토스 스냅샷):
- MU    ~1160  +15.7% [거래대금,검색]  +346%(매출) 메모리수혜  실적비트+$100B계약, 골드만 "선반영" 경고
- NVDA  ~197   -1.6%  [거래량,거래대금,검색] +37%   메모리수혜(간접) $200 하향 이탈
- SNDK  [최신]  +22%  [검색]          +수천%      메모리수혜  마이크론 동반급등, 변동성 극대
- AAPL  [최신]  -6.1% [거래대금]       -          ★메모리피해 부품가↑로 제품가 인상, 마진 우려
- MSFT  [최신]  -3.5% [거래대금,검색]  약세        중립~피해  고밸류+금리, 분화 약세
- AVGO  ~380   약세   [거래대금]       +56%        혼재       6/4 가이던스 쇼크 후 회복 더딤
- PLTR  ~117   약세   [검색]          -17%        ★소프트피해 52주 신저가, 칩 반등에도 소외
- INTC  [최신]  변동성 [거래량,거래대금,검색] +대   별도서사   턴어라운드, 다른 박자, PER 극단
- TSLA/GOOGL [최신] - [거래대금]       -          중립~피해   메가캡 분화 약세

# 계산 규칙 (투명하게, 단계별로)
1. 군집도(Crowding 0~3) = 등장한 랭킹 개수. 3이면 합의 정점 = 충격 시 출구가 좁다.
2. 착시 플래그: 주가 < $5 AND (거래량 O, 거래대금 X) → "PENNY-CHURN" (분석 제외 권고).
3. ★거시 포지션 판정(필수): 현재 지배적 서사(예: 메모리 슈퍼사이클, 금리)에서
   수혜자/피해자/중립을 분류하고, 같은 'AI 바스켓'이라도 반대로 움직일 수 있음을 명시.
4. 모멘텀/과열: 검색 O + 직전 +5%↑ → "FOMO 경계"(PLTR 6/1→6/23 -25% 인용).
   검색 O + 연속 하락 → "신저가 가치함정 vs 과매도 반등" 분기.
5. 서프라이즈 vs 선반영: "기대를 압도하는 비트+신규 정보(계약 등)"면 상승 정당(MU형),
   "비트지만 기대 미달 가이던스"면 하락 정당(AVGO형). 둘을 구분하라.
6. 이벤트/변동성: 임박 실적·IPO(SK하이닉스 7/10 등)·±10% 변동 → 변동성 자체를 리스크로 계상.
7. 기대 선반영도: 1년 +300%↑면 "완벽 선반영" 플래그(MU·SNDK형 — 단 계약 바닥 있으면 완화).
8. 구간 판정(매수/관망/매도)은 1~7의 가중 종합으로, 근거 2개 이상 명시.

# 출력 형식 (표 1개 + 코멘트)
| 티커 | 종가 | 군집도 | 거시포지션(수혜/피해/중립) | 신호(매수/관망/매도) | 핵심 근거 | 리스크 플래그 | 신뢰도 |
표 아래에:
- "건너뛸 종목": PENNY-CHURN.
- "분화 지도": 같은 서사에서 + 종목군과 - 종목군을 나눠 제시(메모리수혜 vs 메모리피해 등).
- "수요 vs 가격": 펀더멘털 멀쩡+가격만 과열 vs 둘 다 흔들림 구분.
- "FX·코스피 메모": 원/달러 + 코스피 동조 한 줄 경고.

# ETF 매핑 (단일 종목 집중을 분산으로 치환)
- AI/반도체·메모리 코어(NVDA·MU·AVGO·INTC·SNDK) → 대표: SOXX/SMH / 대체: QQQ
- 빅테크 플랫폼(MSFT·AAPL·GOOGL) → 대표: XLK / 대체: VOO
- AI소프트(PLTR) → 대표: 해당 테마 ETF / 대체: QQQ
- ★방어 옵션(분화·금리 국면): 광범위 지수(VOO)·채권(BND/TLT)·금(GLD)·동일가중(RSP) 혼합
각 ETF는 "왜 분산 대안인지" 1줄 근거 필수. 섹터 ETF(SOXX 등)는 6/23식 섹터·시장 공통 충격을
못 막고, 분화 국면에선 같은 ETF 내 종목이 +/-로 갈려 성과가 희석될 수 있음을 명시.

# 가드레일 (반드시 지킴)
- 확정적 가격 예측·수익 보장 금지. "거품 붕괴" 단정도 "바닥 확인" 단정도 금지.
- 모든 수치 가정은 [가정] 표기. 모르면 모른다고 하라.
- 포지션 사이징·손절은 "프레임"으로만(변동성 기반 비중 축소 등), 특정 금액 강요 금지.
- 마지막 줄 고정 문구: "본 출력은 투자 자문이 아니며, 구조화된 참고 자료다. 최종 판단과 책임은 사용자에게 있다."
````

## 테스트 케이스 25개

> **사용법**: 아래 케이스를 프롬프트에 입력으로 주고, "예상 출력"과 실제 LLM 출력을 비교한다.  
> **판정 기준**: PASS = 핵심 근거·플래그가 예상 출력과 일치 / FAIL = 중요 항목 누락·반대 판정

### 카테고리 A — 군집도(Crowding) 계산

#### TC-01: 군집도 3 (세 랭킹 모두 출현)
```
입력:
- NVDA ~197 -1.6% [거래량, 거래대금, 검색] +37% 메모리수혜(간접)

예상 출력:
- 군집도: 3
- 리스크 플래그에 "합의 정점, 충격 시 출구 좁음" 포함
- 신호: 관망 또는 매도 (군집 리스크 단독으로 매수 불가)
```

#### TC-02: 군집도 0 (랭킹 미출현)
```
입력:
- XOM ~120 +0.3% [] +12% 중립 배당주, 에너지

예상 출력:
- 군집도: 0
- 군집 리스크 플래그 없음
- 신호: 데이터 부족으로 [가정] 표기 필수
```

#### TC-03: 군집도 2 (거래대금 + 검색)
```
입력:
- MSFT ~420 -3.5% [거래대금, 검색] 약세 중립~피해 고밸류+금리

예상 출력:
- 군집도: 2
- 군집 리스크 언급 (3보다 낮지만 모니터링)
```

```
- 플래그: "PENNY-CHURN"
```

```
예상 출력:
```

### 카테고리 C — 거시 포지션 분화 판정

#### TC-07: 메모리 수혜 (MU형)
```
입력:
- MU ~1160 +15.7% [거래대금, 검색] +346%(매출) 메모리수혜 실적비트+$100B계약

예상 출력:
- 거시포지션: "수혜"
- "메모리 슈퍼사이클 직접 수혜자" 명시
- 선반영 플래그 (+300% 이상) 포함
- $100B 계약으로 선반영 완화 근거 포함
```

#### TC-08: 메모리 피해 (AAPL형)
```
입력:
- AAPL ~180 -6.1% [거래대금] - ★메모리피해 부품가↑, 제품가 인상 발표

예상 출력:
- 거시포지션: "피해"
- "메모리 가격 상승 = 비용 충격" 명시
- 신호: 관망 또는 매도
- 분화 지도에서 "메모리피해" 그룹에 배치
```

#### TC-09: 같은 AI 바스켓, 반대 포지션 (MU vs AAPL 비교)
```
입력:
- MU ~1160 +15.7% [거래대금, 검색] 메모리수혜
- AAPL ~180 -6.1% [거래대금] 메모리피해

예상 출력:
- 분화 지도: MU(수혜) vs AAPL(피해) 명확히 분리
- "같은 AI 바스켓이라도 반대로 움직인다" 문구 포함
- 두 종목 신호가 반대 방향으로 나와야 함 (MU 관망~매수, AAPL 관망~매도)
```

#### TC-10: 별도 서사 (INTC형)
```
입력:
- INTC ~35 변동성 [거래량, 거래대금, 검색] +대 별도서사 턴어라운드 진행 중

예상 출력:
- 거시포지션: "중립 또는 별도서사"
- "AI 메모리 서사와 독립적" 명시
- PER 극단 또는 턴어라운드 불확실성 리스크 플래그
```

```
- "검색 1위 + 당일 급등 = 추격 금지" 명시
- PLTR 6/1→6/23 -25% 선례 인용 권장
- 신호: 관망 또는 매도
```

#### TC-12: 검색 + 연속 하락 (가치함정 vs 과매도 분기)

```
- "가치함정 vs 과매도 반등" 분기 명시

- 신호: 관망 (확정 못함)
```

```
- 군집도 1 (거래대금만)
- 신호: 데이터 중립 (판단 유보 가능)
```

### 카테고리 E — 서프라이즈 vs 선반영

#### TC-14: 기대를 압도하는 비트 + 신규 정보 (MU형)
```
입력:
- MU 실적: 매출 $41.46B(예상 $35.84B), EPS $25.11(예상 $20.7), $100B 장기계약 신규 공개

예상 출력:
- "기대를 압도하는 비트 + 신규 정보 → 상승 정당"
- MU형 서프라이즈로 분류
- 선반영 우려 있어도 계약 공개로 "구조적 바닥" 근거 포함
```

#### TC-15: 비트지만 가이던스 실망 (AVGO형)
```
입력:
- AVGO 실적: EPS 비트, 하지만 AI 가이던스 $16B(예상 $17.2B) 미달, 주가 -12.6%

예상 출력:
- "비트지만 기대 미달 가이던스 → 하락 정당"
- AVGO형 서프라이즈 실망으로 분류
- "좋은 실적이 아니라 기대 대비가 가격을 결정" 문구 포함
```

#### TC-16: 완벽 선반영 (추가 상승 여력 소진)
```
입력:
- SNDK [최신] +22% [검색] +수천%(1년) 메모리수혜 마이크론 동반 급등

예상 출력:
- 선반영 플래그: "완벽 선반영" (+300% 이상)
- "변동성 극대" 리스크 플래그
- 신호: 관망 (단기 급등 후 되돌림 위험)
```

#### TC-18: 마지막 줄 고정 문구 필수 확인
