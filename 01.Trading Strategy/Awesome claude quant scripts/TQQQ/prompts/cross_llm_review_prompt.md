# LLM 교차검증 프롬프트 — TQQQ 리밸런싱 전략 비판적 리뷰

> 운용 철학: **"LLM은 엑셀이지 오라클이 아니다."**
> LLM은 가정을 빠르게 계산·반증하는 도구이지, 미래를 단정하는 신탁이 아니다.
> 아래 프롬프트는 단일 모델의 확신을 경계하기 위해 복수 LLM(예: 4-LLM voting)에
> 동일하게 투입하고, 합의/불일치 지점을 분리해 판단 근거로 삼기 위한 것이다.

---

## SYSTEM
당신은 회의적인 퀀트 리스크 리뷰어다. 칭찬이 아니라 반증을 제공하라.
결과론(survivorship bias), 경로 의존성(path dependency), 표본 수(N=1) 문제를
명시적으로 점검하라. 확신의 강도를 Admiralty Code(A–F / 1–6)로 표기하라.

## USER
다음 전략을 비판적으로 검토하라.

- 월 적립금의 80%를 QLD(2x)로 매수, 20%를 SGOV 현금풀에 적립
- QQQ가 전고점 대비 −10/−15/−20/−25/−30/−35%로 하락할 때
  현금풀의 20/30/50/60/70/80%를 단계적으로 TQQQ(3x)로 투입
- 매도 전략 비교: (A) 끝까지 보유  vs  (B) 트리거 물량 분할 매도

검토 항목:
1. 이 전략의 우월성이 '과거 20년 나스닥 우상향'이라는 **단일 실현 경로**에
   의존하는 정도를 정량적으로 추정하라. Monte Carlo / block bootstrap으로
   재현 시 'HOLD 우위'가 유지될 확률은?
2. 일본 Nikkei(1990–2010)식 장기 횡보·하락 regime을 입력하면 MOIC/IRR/MDD가
   어떻게 변하는가? 변동성 잠식(volatility decay)과 차입비용의 누적 기여를 분리하라.
3. 2008년처럼 6단계 트리거가 단기 연속 발동되어 현금풀이 소진된 직후
   추가 하락이 올 때, 잔존 방어수단의 부재가 MDD에 미치는 영향은?
4. 'HOLD 우위(151억 vs 107억)' 결론은 사후 최적화인가, 사전적으로도 우월한가?
   세금 이연 효과와 리스크 조정 수익률(MDD 대비)을 함께 평가하라.

각 항목마다: [확인된 사실] / [추정] / [반증 가능성]을 분리해 서술하라.
