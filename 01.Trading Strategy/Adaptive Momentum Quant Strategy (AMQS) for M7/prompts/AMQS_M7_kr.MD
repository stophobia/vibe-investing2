# AMQS-M7: Adaptive Momentum Quant Strategy for Magnificent 7 (한국어판)

> **AMQS의 M7 특화 확장판** — *4-Factor Momentum Composite* 에 *Pullback-in-Uptrend (단기 하락 매수 모멘텀)* 차원을 추가하여, *M7 7종목 한정 동적 포트폴리오 운용*. 원본 AMQS의 거시 레짐 필터·-12% 손절·주간 리밸런싱 규칙을 그대로 계승.

## 사용 방법

1. 아래 프롬프트 블록을 *전체 복사*
2. Claude / GPT-5 / Gemini / DeepSeek 등 LLM 에 *그대로 붙여넣기*
3. LLM 이 *M7 7종목의 점수·신호·비중·백테스트* 자동 생성

> **본 프롬프트는 [AMQS 원본 프롬프트](https://github.com/gameworkerkim/vibe-investing/blob/main/01.Trading%20Strategy/Adaptive%20Momentum%20Quant%20Strategy%20(AMQS)/Momentum%20Quant%20Prompt%20kr.MD) 를 확장한 sub-strategy 입니다.** 차이점: (1) Universe 가 *NASDAQ-100+AI Value Chain ~150종목* 에서 *M7 7종목으로 축소*, (2) *5번째 채점 차원으로 단기 하락 매수 모멘텀 (15%) 추가*, (3) Top-10 선별 대신 *7종목 score-tilted allocation*.

전제: *추세는 흔들릴 때 가장 강하게 매집해야 한다.* 단순 모멘텀 전략은 "단기 하락 = 매도" 로 처리하지만, *장기 추세가 살아있을 때의 단기 조정은 가장 비대칭적인 매수 기회* 입니다.

신호 활성 조건 (모두 충족 시에만 점수 계산, 그렇지 않으면 0점):

   (iv) 5일 수익률 ≤ -3% OR 20일 수익률 ≤ -5%  ← 의미 있는 단기 하락

신호 강도 산출:
   기본 = (0.7 × |ret_5d| + 0.3 × |ret_20d|) × (1 + min(factor_A, 1.0))
   + RSI(14) ≤ 30 가산점 (과매도 확인)
   + 52주 신고가 대비 -5%~-15% 구간일 때 추가 가산 (얕은 조정 우선)

z-score 정규화 후 100점 만점으로 변환.

(선택적) Kelly 1/4 적용:
- f* = ((총점 - 50) / 100) / 변동성²
- 보수적 적용: f*/4

### DIP_BUY 신호의 4중 안전장치

원본 AMQS 가 "12-1 모멘텀 > 0" 만으로 추세 판정을 했다면, AMQS-M7 의 DIP_BUY 는 **4개의 동시 조건**을 요구:

| 조건 | 이유 |
|---|---|
| 12-1 모멘텀 > 0 | 학술적 모멘텀 시그널 유효 (Jegadeesh-Titman) |
| 6-1 모멘텀 > 0 | 중기 추세도 함께 살아있음 (cross-validation) |
| 현재가 > 50DMA | 기술적 추세 미붕괴 (단순 50일 평균선) |
| 5D 또는 20D 의미 있는 하락 | 진짜 조정 (노이즈 아닌 의미 있는 폭) |
