# 한계기업 공시 관계망 추적: 가설 검증 문서

> **핵심 가설**  
> 코스닥 시총 500억 미만 한계기업의 공시 이벤트에 반복적으로 등장하는 자연인·법인·조합 노드의 결합 패턴은, 해당 기업의 후행 6~12개월 내 감사의견 거절·관리종목 지정·상장폐지 확률을 유의하게 예측한다.

| 항목 | 값 |
|---|---|
| 문서 버전 | v0.1 |
| 작성일 | 2026-05-12 |
| 대상 시장 | KOSDAQ |
| 모니터링 풀 | 시총 500억 원 미만, 100종목 |
| 집중 대상 | 그 중 단기 거래량 spike 상위 10종목 |
| 정량 가설 수 | 3 (C1, C2, C3) |

---

## 0. 요약

가설은 **방향성으로 정확**하다. 한국 자본시장에서 "한계기업 표적 → 무자본 M&A → CB/BW 발행 → 주가부양/매도차익 → 감사의견 거절/상장폐지" 사이클은 언론과 학계 양쪽에서 이미 합의된 관찰이다. 단, 단일 공시 이벤트는 정상기업도 다수 수행하므로 위양성을 줄이려면 **신호의 결합(conjunction)을 모델링**해야 한다. 가장 큰 비기술적 리스크는 **명예훼손·신용훼손**이고, 이는 "사실 색인이지 평가가 아니다"라는 원칙으로 설계 단계부터 방어해야 한다.

---

## 1. 가설의 정당성

가설이 묘사하는 패턴은 한국 자본시장 실무에서 이미 정형화된 관찰이다. 골격은 다음과 같다.

```
한계기업(3년 연속 적자, 시총 작음) 표적화
   ↓
무자본 M&A (SPC·차입·인수대상 자산 활용)
   ↓
최대주주 변경 + 임원진 교체
   ↓
CB / BW / 제3자배정 유상증자 발행
   ↓
사명 변경 + 신사업 추가 (2차전지·AI·로봇·바이오 등 테마)
   ↓
주가 부양 + CB 전환 + 매도차익 실현
   ↓
감사의견 거절 / 관리종목 / 상장폐지
```

이 사이클에 **반복 등장하는 자연인·SPC·신기술조합**이 존재한다는 것이 본 서비스의 추적 대상이다.

## 2. 실증적 근거

### 2.1 정형화된 약탈 사이클: 보도 패턴

#### 2.1.1 셀레스트라 → 브랜드리팩터링 → 동성제약 (2025)

동일 자연인이 운영하던 코스닥사가 감사의견 거절로 상장폐지 심사 절차를 밟고 있는 상태에서, 별도 회사를 통해 다른 중견사를 인수한 사례. 셀레스트라는 15대 1 감자를 단행해 발행주식수를 93% 이상 줄여 기존 투자자에게 직접 손실을 입혔다. 동일 자연인 → 다수 한계기업이라는 "**자연인 노드의 다중 등기**" 패턴이 그래프 1-hop에서 즉시 잡힌다.

출처: 중앙이코노미뉴스 2025-08-22

#### 2.1.2 협진 → 씨아이테크 → 앤로보틱스 (2026)

CB 발행, 매각, 재투자가 순환하는 구조로 실제 현금 유입 없이 지배구조만 강화하는 거래가 반복된 사례. 그래프상 "**동일 인수자 → CB 발행자 → 매도자**" 노드가 폐쇄 루프를 형성하는 패턴.

출처: 머니S 2026-01-08

#### 2.1.3 엔켐 → 플루토스 (전 리더스기술투자) (2025)

콜옵션으로 회수한 CB를 과거 무자본 M&A 이력이 있는 신기사에 취득가 대비 15% 할인 매각한 사례. 정상기업과 의심 거래의 결합이라는 점에서 단순 라벨링은 위험하지만, **CB 인수자의 과거 이력 노드**가 명백한 risk flag임을 보여준다.

출처: 뉴스톱 2025-12-30

#### 2.1.4 마제스타 → 골든레인 → 엠제이비 (KCI 등재 논문)

외부 차입금이 아닌 인수 대상 기업의 자산을 사용한 무자본 M&A가 결국 상장폐지로 이어진 사례를 재무제표·감사의견 데이터로 분석.

출처: 한국학술지인용색인 (KCI)

#### 2.1.5 거시적 패턴 (피치원 2017)

최근 4~5년 사이 3년 연속 적자 코스닥 상장기업을 집중적으로 적대적 M&A로 확보한 뒤 주가조작·재매각으로 수백억 원을 챙긴 후 관리종목 지정·상장폐지로 이어진 "**우량기업이지만 약탈된**" 사례가 수십 건에 달함을 보도. 이는 가설이 묘사하는 사이클이 단발이 아니라 **시장 구조적 현상**임을 입증.

### 2.2 학술적 근거

| 출처 | 핵심 발견 |
|---|---|
| 무자본 M&A 규제 필요성 연구 (KCI) | 인수 대상 자산을 이용한 무자본 M&A가 상장폐지·투자자 피해의 주요 경로 |
| 상장폐지기업 이익관리 특성 연구 (조선대) | 상장폐지기업은 폐지 직전 이전 연도까지 이익을 상향조정하다가 직전 연도에 급격히 하향. 즉 **회계 시그널은 lagging**이며 공시 이벤트가 leading indicator가 될 수 있음 |

## 3. 정량적 단언 (Testable Claims)

가설은 백테스트 가능하도록 세 개의 정량적 단언으로 분해한다.

### C1. 반복 등장 자연인 (Repeat Director)

> 자연인 *p*가 직전 *k*년 이내 *N* ≥ 2개의 한계기업(시총 500억 미만 코스닥) 또는 상장폐지 이력 기업에 등기임원이었다면, *p*가 새로 등기되는 기업 *c*의 12개월 후 관리종목 지정 또는 감사의견 거절 확률 P(*c* → 부실 | *p*)는 베이스라인 P(*c* → 부실) 대비 통계적으로 유의하게 높다.

```
H0: P(c → 부실 | p) = P(c → 부실)
H1: P(c → 부실 | p) > P(c → 부실)
유의수준: α = 0.05
```

### C2. 최대주주변경 + CB + Pivot 결합 (Conjunction Signal)

> 어떤 기업 *c*에서 180일 윈도우 내에 다음 세 이벤트가 모두 발생하면, 24개월 후 상장폐지·감사의견 거절 확률이 베이스라인 대비 유의하게 높다.

```
E1: 최대주주변경
E2: CB 또는 BW 발행
E3: 사명 변경 OR 사업목적 추가 (특히 2차전지/AI/로봇/바이오)
```

### C3. CB 인수자 재등장 (Acquirer Recurrence)

> CB 인수자 *a* (자연인 또는 SPC, 또는 그 실질지배자)가 직전 *k*년 이내 다른 한계기업의 CB 인수 이력이 있으면, 발행기업의 후행 12개월 관리종목 지정 확률은 베이스라인 대비 유의하게 높다.

## 4. 가설의 한계

### 4.1 단일 이벤트의 위양성

CB 발행, 사외이사 변경, 사명 변경, 신사업 추가 — 각각만 보면 정상기업도 다수 수행한다. 위양성(false positive)이 심각하다. 진짜 시그널은 **conjunction**이다.

### 4.2 차명·SPC 체인의 불완전 관찰

한국에서 진짜 실소유주는 신기술조합·투자조합·명의신탁·해외 SPC를 거쳐 가려져 있다. DART 공시만으로는 그래프의 깊이 1~2단계까지는 정확하지만, 그 너머는 추정에 의존한다. 신용평가사 데이터, 법원 등기, 언론 보도와의 cross-reference가 필요하다.

### 4.3 선택편향

어떤 자연인이 "여러 한계기업 등기임원"이라는 사실은 (i) 기업사냥꾼이라서일 수도 있고, (ii) 부실기업 정리 전문 회계사·변호사이기 때문일 수도 있다. **역할(role) 차원**과 **시간 차원**(부임 시점이 부실 발현 전인가 후인가)을 같이 모델링해야 한다.

### 4.4 Lookahead Bias

백테스트 시 "이 사람이 X사 등기임원이 된 시점에 그 정보가 공시되었는가"를 엄격히 따져야 한다. 임원 정보가 사업보고서(분기)에야 반영되는 경우와 임원변경 공시(즉시)는 다르다.

### 4.5 Survivorship 데이터 접근성

상장폐지된 기업의 과거 공시는 DART에 보존되지만, 그 시점의 시세·거래량은 KRX에서 일부 제거되는 경우가 있다. 백테스트용 가격 데이터는 별도 보존이 필요하다.

## 5. 신호 강도 분류

DART OpenAPI 및 KIND에서 직접 뽑을 수 있는 신호를 강도별로 분류한다.

### High signal

- 주요사항보고서: 전환사채권발행결정, 신주인수권부사채권발행결정
- 주요사항보고서: 최대주주변경
- 감사보고서: 감사의견 거절·한정
- 채권자보전조치, 회생절차개시신청
- 정정공시: CB 전환가액 조정 (refixing)
- 임원·주요주주 특정증권등 소유상황보고서: 단기 매매·집중 매도

### Medium signal

- 임원변경 공시: 사외이사·감사 동시 다수 사임
- 사업보고서 내 "사외이사" 필드 변화율
- 타법인 출자/취득 공시 (특히 비상장 적자기업)
- 신사업 추가 + 사명 변경 + 액면분할 동시기

### Context signal

- 공시 정정 횟수
- 감사인 변경 빈도
- 소액공모 / 제3자배정 유상증자 반복
- 자본금 1,000만 원 수준 SPC가 인수자로 등장

## 6. 백테스트 설계

### 6.1 정답 표본

최근 5년(2021~2025) KIND 기준 상장폐지·관리종목 지정 종목 중 **시총 500억 원 미만 코스닥** 필터로 추출. 목표 표본 수: 100~150건.

### 6.2 매칭 표본

같은 시기 · 같은 시총 구간 · 같은 업종이지만 정상 유지된 기업 매칭 표본 300건. 1:3 매칭.

### 6.3 윈도우

상장폐지·관리종목 지정 6개월 lookback에서 신호 발생 여부를 측정.

### 6.4 메트릭

| 메트릭 | 목표 |
|---|---|
| Single-rule AUC | ≥ 0.60 |
| Conjunction AUC | ≥ 0.80 |
| Precision @ k=10 (월간 알림 상위 10) | ≥ 0.40 |
| Recall @ 6개월 | ≥ 0.55 |
| False alarm rate (정상기업 신호 발생률) | ≤ 0.10 |

### 6.5 검정 절차

1. 정답 표본과 매칭 표본을 합쳐 패널 데이터 구성
2. 각 룰을 개별 적용해 단일 신호 AUC 측정
3. Conjunction (룰의 논리곱·가중합) AUC 측정
4. McNemar test 또는 DeLong test로 단일 vs 결합 신호 차이 검정
5. 1년 holdout 기간으로 out-of-sample 검증

## 7. 윤리·법적 가드레일

### 7.1 원칙

자동화 시스템이 "X는 기업사냥꾼"이라는 **평가적 라벨**을 생성·표시하면 다음이 동시에 발생할 수 있다.

- 형법 제307조 명예훼손 (사실적시·허위사실적시)
- 형법 제313조 신용훼손
- 정보통신망법 제70조
- 자본시장법상 풍문 유포 가능성

### 7.2 방어 가능한 설계 6원칙

1. **사실(공시 원문)만 표시하고 평가는 사용자에게 위임.** "이 자연인이 등기임원이었던 회사 중 N개가 상장폐지 이력을 가짐"은 사실 적시. "이 사람은 기업사냥꾼임"은 평가. **전자만 출력**.
2. **자동 라벨링 금지.** "위험인물 점수 8.7" 같은 단일 스코어는 자연인 단위로는 표시 금지. 종목 단위로만 점수 표시.
3. **소스 백링크 의무.** 모든 데이터 포인트에 DART rcept_no 또는 원본 URL 백링크.
4. **언론보도 인용은 따옴표·출처.** 자체 주장으로 재구성하지 않음.
5. **약관에 "투자 권유 아님, 보도 보조 자료" 명시.**
6. **사용자 제한.** retail 공개보다 언론사·증권사 컴플라이언스·VC 실사 등 B2B 채널 우선.

### 7.3 사용자 자격

본 서비스는 **언론사 등록 도메인 이메일로 인증된 사용자만** 가입 가능하도록 설계한다. 이는 (i) 일반 retail 노출 회피, (ii) 공익적 보도 목적 방어 논리 확보, (iii) 사용자 책임 분산을 위한 조치다.
