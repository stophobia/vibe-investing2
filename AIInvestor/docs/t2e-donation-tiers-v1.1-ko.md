## 13. 미니앱 온보딩 → 도네이션 진입 동선 (신규)

### 13.1 진입점 통합

dev-spec v1.1 §12.2에 정의된 4단계 온보딩 플로우 중 **Step 2 (오늘의 운세 + 3종목 추천)**에서 잠금 종목 카드를 클릭하면 도네이션 화면으로 자연스럽게 연결한다.

```
[Step 2 화면]

  ┌────────────────────────────────────────┐
  │  ✨ 오늘의 행운 번호: #7                │
  │  ──────────────────────────────       │
  │  💚 안정형 (Low)   - 🔓 무료 추천      │
  │     AAPL  $165.30 ▲                    │
  │     [상세 분석 보기]                    │
  │                                        │
  │  💛 성장형 (Medium) - 🔒 잠금          │
  │     [100 P 또는 후원으로 해금]          │
  │                                        │
  │  ❤ 공격형 (High)   - 🔒 잠금          │
  │     [100 P 또는 후원으로 해금]          │
  └────────────────────────────────────────┘

  잠금 카드 클릭 시 ↓

  ┌────────────────────────────────────────┐
  │  🔓 종목 해금하기                       │
  │  ──────────────────────────────       │
  │  방법 1️⃣  포인트로 해금                 │
  │     ▷ 100 P 차감 → 즉시 해금            │
  │                                        │
  │  방법 2️⃣  USDT 후원으로 해금             │
  │     ▷ $10 후원 → 300 P 지급             │
  │       (3개 종목 모두 해금 가능)          │
  │     ▷ $50 후원 → 2,000 P 지급           │
  │       (20일치 잠금 종목 해금분)         │
  │                                        │
  │  방법 3️⃣  친구 초대로 해금               │
  │     ▷ 1명 초대 + 가입 완료 시 즉시 해금 │
  │                                        │
  │  [후원하기]  [친구 초대]  [취소]         │
  └────────────────────────────────────────┘
```

### 13.2 잠금 종목 해금 정책

| 채널 | 비용 | 해금 범위 | 효과 |
| --- | --- | --- | --- |
| 포인트 | 100 P | 1개 잠금 종목 1회 | 즉시 |
| USDT 후원 | $1+ | 누적 P로 환산 후 차감 | 즉시 + 명예 등급 가산 |
| 친구 초대 | 1명 verified | 1개 잠금 종목 1회 | 친구가 §12 Step 1 (생년월일) 완료 시 |

> **주의**: 친구 초대 채널은 dev-spec v1.1 §16.1 부정 방지 규칙 적용 — 30일 내 활성도 ≥3일 미만 유저는 무효.

### 13.3 무료 종목 변경 정책

- 하루 1번 결정론적 선정 (§12.3 of dev-spec v1.1) — 같은 날 변경 불가
- 단, **$1 이상 후원자**는 다음 행운 번호로 1회 재추천 권한 (`fortune_reroll`)
- 누적 후원 $100+ Supporter는 매일 재추천 1회 무료 + 무한 시도 (단 동일 lucky_number 재출현 제한)

### 13.4 신규 엔드포인트

| Path | Method | 용도 |
| --- | --- | --- |
| `/api/fortune/unlock` | POST | 잠금 종목 해금 (channel: points/donation/invite) |
| `/api/fortune/reroll` | POST | 운세 재추천 (Supporter 등급 이상) |

## 14. 도네이션 ↔ 추천인 랭킹 결합 (신규)

dev-spec v1.1 §16.1에 정의된 추천인 랭킹 점수 공식에 도네이션 가산점이 이미 포함되어 있음:

```
referrer_score(R) = sum_over_referees(
    1.0           if referee is verified
  + 0.5           per active_day of referee in last 30d
  + 2.0           if referee made any donation     ← 도네이션 가산
  + 0.2           per referee_prediction_settled
)
```

본 문서에서는 이 점수가 추천인 자신에게 주는 추가 도네이션 인센티브를 정의:

### 14.1 추천인 도네이션 보상

| 피추천인의 도네이션 누적 | 추천인이 받는 추가 보상 |
| --- | --- |
| $100+ (Supporter) | 추천인에게 +500 P 일회성 |
| $500+ (Patron) | + 다음 1개월 프리미엄 1회 무료 |
| $1,000+ (Benefactor) | + 영구 골드 티어 보장 |
| $5,000+ (Champion) | + 영구 플래티넘 + 1:1 운영자 미팅 1회 |

### 14.2 부정 방지

- 동일 device fingerprint × IP 클러스터로 자가 추천 + 자가 도네이션 시 점수 0
- 추천인의 추가 보상은 피추천인 도네이션 후 **30일 holding period** 후 지급 (분쟁/환불 분쟁 흡수)
- 단, 도네이션은 환불 불가이므로 holding period는 운영 분쟁 (사기/계정 도용)만 흡수
