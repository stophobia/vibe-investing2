개인정보 처리방침

최종 개정: 2026-05-07 · Last updated: 2026-05-07

핵심 고지: 본 서비스는 사용자의 검색·예측·사주 입력 데이터를 익명화한 형태로 분석 모델 학습 및 학술 연구 논문에 활용할 수 있습니다. 거부하시는 경우 봇에 /forget 명령으로 검색 기록 삭제를 요청하거나 /forget 후 사용을 중단하시면 됩니다.

1. 수집하는 정보

  • 식별 정보: Telegram user ID는 SHA-256 해시(salt 포함)로 변환하여 16자 익명 ID로만 저장합니다. 원본 텔레그램 ID는 저장하지 않습니다.
  • 활동 로그: 검색 종목, 페르소나 선택, 예측 제출, 사주 생년월일시, 포인트·티어 변동.
  • 언어/지역 설정: UI 언어 (ko/en/ja/zh).

2. 정보 활용

  • 서비스 제공 및 개선 (캐싱, 추천, 매치업 매칭)
  • 개인 검색 정보 로그를 익명화하여 분석 모델 학습, 제3자(연구 협력 기관) 제공, 학술 연구 논문에 활용 가능합니다.
  • 익명화 절차: 사용자 식별자는 SHA-256(salt) 해시로만 노출되며 salt는 주기적으로 회전(rotation)됩니다. 동일 사용자도 시점이 다르면 추적이 어렵게 설계되었습니다.

3. 거부 및 삭제 요청

위 익명화 데이터 활용을 거부하시는 경우:

  • 봇에서 /forget 명령으로 검색 기록 및 사주 입력 데이터의 삭제를 요청할 수 있습니다.
  • 탈퇴(서비스 사용 중단)하시면 추가 데이터가 수집되지 않습니다.
  • 삭제 요청은 즉시 처리되며, 이미 익명화되어 연구 데이터셋에 포함된 과거 기록은 식별 불가하므로 회수가 어려울 수 있습니다.

4. 보관 기간

  • 활동 로그: 12개월 (이후 자동 삭제 또는 익명화)
  • 사주 입력: 사용자가 직접 삭제 요청하기 전까지 보관
  • 익명화된 통계/연구용 데이터: 무기한 (개별 식별 불가)

5. 제3자 제공

  • 법령에 따른 요청이 있는 경우
  • 익명화된 형태로 학술 연구 협력 기관에 제공
  • 그 외에는 제공하지 않습니다.

6. 보안

모든 데이터는 Azure Blob Storage에 암호화 저장되며 (at-rest encryption), 전송은 HTTPS로 보호됩니다. Telegram WebApp init_data는 HMAC-SHA256으로 검증합니다.

7. 사용자 권리

  • 본인 데이터 조회·수정·삭제 요청권 (/forget)
  • 익명화 활용 거부권
  • 탈퇴권

8. 면책

본 서비스의 사주 분석, 종목 추천, 매치업 예측, 페르소나 분석은 모두 참고용이며 투자 자문이 아닙니다. 모든 투자 판단의 책임은 사용자 본인에게 있으며, 운영자는 투자 결과에 대한 책임을 지지 않습니다.
